# Sales Forecasting & Demand Intelligence Dashboard

## Project Overview

This project predicts future sales using multiple forecasting models and provides business insights through anomaly detection, product demand clustering, and customer segmentation.

## Features

- SARIMA Forecasting
- Prophet Forecasting
- XGBoost Forecasting
- Model Comparison
- Category & Region Forecasting
- Isolation Forest Anomaly Detection
- Z-Score Outlier Detection
- Product Demand Clustering
- Customer Segmentation (RFM + K-Means)
- Streamlit Dashboard

## Best Model

SARIMA

MAE : 19244.49

RMSE : 19950.07

MAPE : 20.53%

- Python
- Pandas
